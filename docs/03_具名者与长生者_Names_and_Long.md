# 具名者

> “我会给你签名。但我自己还并具名。还不行，大概永远都不行。”
>
> —— “巴尔多梅人”特蕾莎·加尔米耶

具名者是漫宿现有秩序体系下的一种超然存在，其位格在长生者之上、司辰之下。具名者通常具有强大的力量，是某种法则某些方面的化身，可以根据授权代行权柄，其举动能对漫宿和醒时造成不容小觑的影响。

具名者在秘史世界的神学里，通常代表司辰权柄的某种侧面，当与其所侍司辰并提时，即作为其面相出现。依照传统理论，每位司辰对应七位具名者——正所谓“皆具七名”，这种意义上的具名者与其主关系密切，以至于难解难分，甚至可以被称作某种“自我”。
具名者亦可视为因具有某种力量而占据此阶位格的存在，因其保有力量、并非依赖某一司辰分授权柄，因而并非与其主一俱荣损。

具名者有时也被称为“次级神明”（minor deity）或“小神”（little god）

也许值得一提的是，名字在秘史世界与我们的历史中的某些神秘主义中具有神秘力量，“世间万物唤名皆至”。

## 孰具其名

在密教模拟器中，玩家在飞升为长生者后可以通过达成一系列条件举办令自己飞升为具名者的仪式。这些仪式的核心要义可以概括为：向目标司辰展示自己在完成司辰工作目标的能力和意愿，例如复现当年司辰所成之举，从而谋求晋升。一般来说，“遣汝大能，视吾伟业！”这样豪迈的句式并不适用于具名者应聘。

## 具分类名

我们可以笼统地按准则将具名者称为某准则之具名者，亦可以其具名之具体名类称呼，或者指名某具称其名的存在时具称其名。

赤杯的具名者被称为干渴者、骨白者与美貌者，可能出于赤杯具名总是相互吞噬或吞噬他人而难有固定人选，在称呼他们时很少直指其名。

# 丽姬亚

**[万登林]（[留言]） 2025年2月5日 (三) 16:57 (CST)**

> 我们中的一位通过顶点，另一位或许没有；  
> 两位觅得**灯**，两位选择逃离；  
> 第七位的名字鲜被提及，但从未被遗忘。
>
> —— [关乎钥匙]

丽姬亚——“持钥匙者”——共有七位，她们皆犯下天孽，因而被居屋的法则排除在外，也因而得以持有并使用[钥匙]。尽管身犯天孽，丽姬亚依然在居屋的坐席内享有尊贵地位——她们皆是双生子的具名者，因双生子结合不应分离之物，而“天孽”在某种意义上是至高无上之结合。

丽姬亚们“有着不竭的食欲”，她们目前的存在方式正是作为[阿卢卡]——甚至可以说，她们本身属于最高级的吞噬者。

丽姬亚并不总是那七位，但总是有七位，正如居屋的[钥匙]并不总是那样的七把，但总是有那七把。目前（截止至新作《夜游漫记》公布发售日期前），七位丽姬亚分别是：[镰齿族]遗民美杜莎，通晓光的[浪游旅人]之友施鲁塞尔女士，[七蟠]前具名者“怪物之母”厄客德娜，兜颈贵胄之女与丁香烟爱好者苏洛恰那·阿摩伐舍，社交名媛“诞于海、逐于潮”者摩根勒菲，年轻且食欲惊人的“操线仙娥”玛丽内特，噤声居屋的守护人“开海者”罗威娜。她们各自所持钥匙见[钥匙与门]页面。

丽姬亚在醒时世界有时以一个松散小团体活动，如《维也纳未解之谜》时代的“新生丽姬亚”和如今的丽姬亚俱乐部。至少一次，她们包下了赫柏·斯坦顿号。

可能地，美狄亚是某位美杜莎之一。
